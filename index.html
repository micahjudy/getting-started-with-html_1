Internet

The internet works by useing its own unique language called HTTP. The process starts by you opening up a browser like Google
the browser then sends a request to the server finds the HTML document and sends it back to the browser.
The browser then interprets that information and desplays in on the users screen.

HTML

HTML documents are what most on the web is madeup of.
HTML stands for Hypertext Markup Language.
It contains text that make up what you see and links to documents
that make up how you see it as well.

Tags

Tags are what HTML uses to distinguish how browers react to the document.
Example of some tages
Bold-b
Paragraph-P
Images-IMG
Tags are also madeup of either Inline or Block .
Block tags have something decribed as an "Invisible Box" surrounding the contents inside the tag.
